# Ponto&Vírgula – Resenhas de Livros

Este site foi desenvolvido **exclusivamente para fins acadêmicos**, como parte da disciplina de **Programação de Aplicações Web**.  
O objetivo do projeto é praticar conceitos de **HTML, CSS, JavaScript e organização de um site estático**, incluindo navegação entre páginas, estilização e estruturação da coleção de livros de forma organizada, visual e acessível, permitindo a navegação entre títulos e a pesquisa por nome, sem a utilização de banco de dados.

---

## Sobre o Projeto

O **Ponto&Vírgula** simula um site de resenha digital onde os usuários podem visualizar capas de livros, ler resenhas com opiniões sobre a publicação, acessar páginas individuais de cada obra e utilizar uma barra de pesquisa para localizar títulos específicos.

**Importante:**  

- Os **textos contidos no site NÃO são autorais**.
- Cada livro possui a indicação de seu respectivo autor ou detentor dos direitos.  

Este projeto tem finalidade **exclusivamente educacional**.

---

## Tecnologias Utilizadas

- HTML
- CSS  
- JavaScript
- Google Fonts  
- Estrutura de site estático (sem banco de dados)
